# TableMaker

## 項目簡介
TableMaker 是一個用於創建和管理表格的網頁應用程式。用戶可以自定義表格的列數、行數、字型、邊距等，並生成表格。這個應用程式可以幫助用戶快速生成格式化的表格，並支持保存和讀取表格數據。

## 功能特性
- 自定義表格的列數和行數
- 設定字型和邊距
- 支持多種排列方式和書寫方向
- 即時顯示資料的數量和表格頁數
- 保存和讀取表格數據

## 安裝和使用
1. 克隆此儲存庫：
    ```bash
    git clone https://github.com/your-username/TableMaker.git
    ```
2. 打開`index.html`文件以使用應用程式。

## 使用指南
1. **資料輸入**：
    - 在文本區域中輸入您的表格數據。
2. **欄位和行數**：
    - 設定表格的欄位數和行數。
3. **邊距**：
    - 設定水平和垂直邊距。
4. **字型和分隔符號**：
    - 設定表格的字型和資料分隔符號。
5. **排列方式和書寫方向**：
    - 選擇表格的排列方式和書寫方向。
6. **生成表格**：
    - 點擊“產生”按鈕生成表格。
7. **保存和讀取數據**：
    - 使用“保存”按鈕保存數據，使用“讀取”按鈕讀取數據。
